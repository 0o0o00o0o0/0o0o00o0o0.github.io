我的国企之旅

阮三丰 2021-04-21 21:21:31 生活杂文

# 前言

时间来到2020年12月。新组长统一，老部下退位，我也慢慢感受到我不能呆下去了。于是打开了招聘软件，机缘巧合下我看到了长沙某大型国企在招web前端。于是思考了很久，投递了一份。

# 面试

同月面试。那天是一个星期五的上午，同时有12位竞争者，我是第9个。轮到我的时候已经快到了将近12点，面试官就问了很简单的vue的问题，然后上机写一个布局就要我走了。也算是乌龙，我快到家的时候又要我会去面试，说没面试完。我又屁颠屁颠的回去面了。主要问了国企的利益和个人利益的权衡的问题，和我的期望薪资。我说了JD上写的范围内的薪资，然后人事默认，面试正式完成。

# 录取

2月份结果我通过了面试。hr老师同我交流了一波，我的底薪为几千，然后绩效项目奖一年大约多少多少万，我折合一算，一年下来也将近十七八万，还不错。就答应了，然后offer也通过邮件发给我了，只不过是一张照片.....

# 报道

3月1号。我正式报道并入职了这家国企。福利不错，每个月报销一定金额的油票，午餐6元4菜1汤1水果，过年过节都发钱。我也想快点适应这份工作，毕竟国企工作机会是很难得的。然后发现上次9位对手全被我PK掉了，只有我来入职。我很奇怪也很庆幸。

# 适应

整个3月份我都是没又做项目的。原因很简单，第一个月是没有绩效没有项目奖金的，所以组长干脆不分配我工作。部门内项目我也仅仅接触了一个，是用vue写的。可能以后主要使用vue开发吧，那么我的ts,react可能难以大展身手，我当时这么想着。同时进部门的后端开始接触国企合作的第三方厂家的代码，具体过程不便赘述。我也算是开始接触国企内部的项目模式。

# 转变

心态转变就是第一个月工资。我知道应该不多，但是确实低的我不敢相信。原来第一个月是有绩效的，同人事聊的基本底薪为几千,现在基本底薪加上绩效也才这么多，然后八折。我平时活跃的后端新同事也默然了。虽然很低落，但是心存念想，我可以通过多做项目获取项目奖金来提升自己薪资。至少不要比上一家难看。

# 落寞

时间来到了4月份。项目确实越来越多了，可是项目的归属就跟迷一样。今天听说一个几百万的大项目我们国企接了准备自己做，第二天领导一拍板，外包给第三方做。于是乎我今天很开心可以做项目了，一定要好好主导认真搭好架构，到了第二天发现自己白高兴一场。关键是这种情况很多，我对于项目的憧憬也越来越渺茫。我们组也总是被领导批判，挨骂我们总是一脸茫然的带头冲锋。我们组长也分配了一些零星的项目给我，所以整整两个月，我可以说是坐在那儿发呆的。心情渐渐浮躁

# 尾

我今年30了，我一直认为我是到了养老的年纪。进国企可以每天舒舒服服的，不担心被代替。确实我做到了，可是我发现，我还存留着很大的冲劲去创造项目，去创造价值。我身上还有房贷压着我，欠款压着我，我要舒服也舒服太早了。所以我打算离职去重新定位自己的价值。我上个老大是个狠人，他比我大了3岁，却在去年去了阿里巴巴做技术专家，年龄或许真的不一定是一个程序员是否被淘汰的标准，慵懒，做作，踏步不前可能才是最危险的。我并不具备这种性格，我也就一直有着写项目，解决问题的劲儿。

今天是4月21号，明天我就准备去提离职结束我长达2个月的国企休闲时光，重新拿起武器继续战斗吧。
