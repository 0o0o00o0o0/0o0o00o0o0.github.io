关于我在人和未来生物科技有限公司

关于我在人和未来生物科技有限公司

阮三丰 2023-03-11 21:21:31 生活杂文

人和未来生物有限公司(后文用：人和)是我至今职业生涯中最大的幸运和幸福，也是我心中最理想的。双休不加班，工资按时发，打卡弹性制，工作人性化，未来发展不可估量。但，是真正要说再见了。

# 起源

2018年，具备ng4\react和ts能力的，还是萌新的我被裁了，依稀记得原因：美国制裁中兴，我又属于中兴的外包公司，然后，就被裁员了。那时候不懂裁员的措施，稀里糊涂的拿了一点点补偿就走了，还很开心。但是形势所迫，我亦无所谓了。开始努力面试，不过有点懵，因为全是要vue的，而我还不怎么会用。正当我准备学一波VUE再去面试的时候，一个电话响起，说要我去面试，地点是在青竹湖。硬着头皮去的，面试过程其实蛮紧张的，因为我老大哥朱德龙面试我的，一来是慕课网讲师，二来是超高学历，三来是严谨的面试氛围。后来人事问我薪资，我说7k，人事满意的让我回家等消息。后来就入职了这家公司。PS：后来听老大说我是他面试中薪资要求最低的，同时技术和大家又差不多，所以要了我，他可能不知道，那时候我是在我原来5k的基础上加了2k，知道我面试成功了我心里乐开了花。

# 提升

入职后，我心中有了一个想法，以朱德龙（后文用：老大）为目标而努力学习。在人和，我的工作效率是极高的，因为我想表现自己，想要让老大觉得我入职是值得的，就，一种很典型的新人情结。同时把vue狠狠的学了一把，后来公司AI组要做一个基因分析的项目，我独自采用了 vue2 + typescript 好好的秀了一把操作，对，独自，后面我基本和AI组合作，独自为他们开发项目。这里就好好的促进了我的前端学习和能力，后面合作开发的项目不乏：react、umi、qiankun、python、taiwindcss、svelte、docker等等技术，还自己用svg写了图形库，可以说是我的思维大爆发的时候，AI组组长（PS：后来的CTO（B））也很看重我。算法方面，老大也会每月出算法题考核我们，并要我们输出技术文章，合格了，我们就能享受他自己提供的大餐。这个时候是我最快乐最狂的时候，网络上，qq群我都有很"大佬"的question，以至于好多人称我为"三丰大佬"。这个时候我的想法莫过于在人和呆一辈子，以能拿人和的"股份"为后续的职业人生规划。

# 转折

正所谓，天下无有不散筵席。2020年底，老大提离职了。就如同，你奋力朝着目标跑，然后突然目标不在了的失落感。这时候，可以看出老大是很挺我的，他跟CTO（A）提出：要我接任他的组长。然而 ~ 我学历不够，普本而已。虽然确实学历不能证明一个人的编程能力，但是能证明那个人够自律。新组长来了，老大走了。新组长是三一出来的，学历嘎嘎高，我也很期待他的实力。然而~他觉得4人前端少了，又招进来2前端、他的领导能力真的很平庸，解决问题的能力真的很一般，最关键的，他有点装。以至于，我们三个老前端是一个流派，他们三是另一个流派。我觉得没意思，我提离职了，说实话氛围不对了。后来老大知道这档子事情还愧疚过一段时间，我觉得不能怪他，只能说自己的实力不够。

# 反悔

我走后，老前端都依次离职了。在各轮面试之后，我入职了大汉。原因：1，离家近、2，伪双休(大小周，小周去上班的时候等于自由时间，去凑个人数)。理想很丰满，现实很骨感，2021年房地产行业低迷，直接催生了无数项目提前完成的目标。于是乎，各类加班应运而生。可以说，我在2021年加班加残了，也摧残了我这份学习的心。大汉还有各种新增的绩效考核，变相的剥削我的工资和年终奖金。我累了，我虽然能继续加班，但是我不想继续加班了，我虽然已经做了N多的封装来加速项目的开发，但是产品的变脸比翻书还快，项目经理的鳄鱼的眼泪让人生厌。我开始怀念在人和 双休不加班的日子，甚至产生了我要不要联系下人和的前同事问问有没有机会回去的想法。事情很神奇，21年底我准备提离职的时候，有心灵感应般的，CTO（B）突然打电话给我，要我回人和，我毫不犹豫的就答应了，而且回来以后薪资水平也高了一些，此时CTO（A）已经离职。

# 失落

我以为得到了CTO(B)的重视，准备大干一场的时候，发现人和变了：新组长不到一年就走了，他招的人变成了新组长。好在这个组长不装，我愿意和他一起合作，而且他还比较尊重我的一些选择，相处下来比较融洽。可 ~ CTO(B)离职了。我被归并到了信息开发部。我觉得也行，给我事情做让我稳定即可。新冠结束了，公司也就没钱了。其实新冠按道理是有钱的，但是欠人和钱的是各地政府，这就好比肉包子打狗了。2022年底开始裁员，2023年3月7号，我被领导单独叫进办公室，我也被裁了。说实话，有点难受，我认真对待这每一次的开发和维护，为的就是希望能留在这里久一些，但是很无奈，领导说，公司没钱了。为什么会裁我呢？因为现在4个前端，一个单独划分到紧急行动小组中，一个分属于大数据的前端，都是领导心腹，一个是组长，就只能裁我了。其实我想说，紧急行动小组的前端工作无非就是小程序，很难吗？大数据那个项目的源码都是我写的，我离职以后交接给另一个前端的。我其实都可以替代啊，毕竟我实力不说多么大佬，也是很强的。回头想，可能我薪资比他们高吧 ~ 3月7号 下午，前端新组长去参加了公司期权购买大会，有点难受，这可以说是我在人和的职业规划目标。如果当初不离职或许那批欢声笑语中有我呢？毕竟3年功勋奖我也有了，呵呵 ~ 至此，我已然对人和没有太多希望了，我以前有多狂妄现在就有多狼狈。

# 終

重新开始准备面试，多少有点迷茫，2023，前端已死不是开玩笑的。不过实力犹在，希望后面能顺利吧
